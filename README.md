# Simple Task Management System (Laravel 11)

A small single-user Task Management System built using **Laravel 11** and **MySQL**.  
The application allows users to create, view, update, mark tasks as completed, and delete tasks with fields such as title, description, priority, and status.

---

## Features
- Add, edit, delete tasks
- Mark tasks as completed
- Priority management
- Laravel MVC architecture
- Bootstrap UI
